TARANTULLA SUITE

Tarantulla is an Open Source project with extractors by channels, for scraping and crawler of web data.

< Do It Yourself />

  • Tarantulla-Facebook is a module from Tarantulla's main suite, which help you with crawler and scraping of public data on Facebook by selected publishers, we use the Graph API from Facebook. Tarantulla-facebook was developed with Python 3.x and use Pentaho Data Integration - PDI to structure data into relational databases.*

    See project

  • Tarantulla-Twitter is a module from Tarantulla's main suite, which help you with crawler and scraping of public data on Twitter by selected publishers, we use the Timeline API from Twitter. Tarantulla-twitter was developed with Python 3.x and use Pentaho Data Integration - PDI to structure data into relational databases.*

    See project

  • Tarantulla-Youtube is a module from Tarantulla's main suite, which help you with crawler and scraping of links on Youtube by topics. Tarantulla-youtube was developed with Python 3.x and use Pentaho Data Integration - PDI to structure data into relational databases.*

    See project

  • Tarantulla-Web is a module from Tarantulla's main suite, which help you with crawler and scraping data from websites. Tarantulla-web was developed with Python 3.x and use Pentaho Data Integration - PDI to structure data into relational databases.*
